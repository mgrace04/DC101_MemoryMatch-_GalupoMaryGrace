1. **Select Difficulty:** Choose between **Easy** (12 cards), **Medium** (16 cards), or **Hard** (30 cards).
2. **The Reveal:** When the game starts, the cards will flip over for a few seconds to give you a "sneak peek." Memorize as many positions as you can!

4. **Win the Game:** Match all pairs to stop the timer and see your final score in the victory modal.
